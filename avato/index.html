AVATO (Available Volume After Take-Out) software

  • Version in 2D periodic space:

    This software calculates the area contents and the length of the curved outlines of cavities in configurations of spheres. That is, the holes which are not covered by any sphere. More precisely, this is the code described and used in the following publication:

      M. Schindler and A. C. Maggs:
      "Cavity averages for hard spheres in the presence of polydispersity and incomplete data"
      Eur. Phys. J. E 38 (2015) 97
      doi:10.1140/epje/i2015-15097-0

    If you use this software, please credit the author by citing this article.
